# REALTIME CHAT GROUP APP 
Fullstack realtime chat app, build with: 

FRONTEND : 
1. Next Js
2. Chakra UI
3. Redux Toolkit
4. Redux persist
5. Socket.io client

BACKEND : 
1. Node Js
2. Express Js
3. Jsonwebtoken
4. Bcrypt
5. Prisma ORM
6. PostgresQl Database
7. Socket.io
